# Инструкция по запуску и использованию программы шифрования TEA  
**Проект:** tea-encryption-project-lu  
**Автор:** студент Лукьянов Александр Дмитриевич

Данный документ содержит простую и понятную инструкцию
по запуску и использованию программы шифрования TEA.
Инструкция рассчитана на пользователей без опыта программирования.

---

## 1. Что это за программа

Программа предназначена для:
- шифрования текстовых данных;
- расшифрования ранее зашифрованных данных.

В основе программы используется алгоритм TEA  
(Tiny Encryption Algorithm).

Работа с программой осуществляется через графическое окно.

---

## 2. Что необходимо для запуска

Для запуска программы требуется:

- компьютер с Windows / macOS / Linux;
- установленный Python версии **3.9 или выше**;
- доступ к терминалу (Командная строка / Terminal).

---

## 3. Как открыть терминал

### Windows
1. Нажать `Win + R`
2. Ввести `cmd`
3. Нажать `Enter`

### macOS
1. Нажать `Command + Space`
2. Ввести `Terminal`
3. Нажать `Enter`

---

## 4. Проверка Python

В терминале ввести команду:

```bash
python3 --version

Если Python установлен, появится сообщение вида:
Python 3.11.0
Если команда не работает, необходимо установить Python с официального сайта: https://www.python.org/

5. Подготовка проекта
1. Скачать проект tea-encryption-project-lu с GitHub.
2. Распаковать архив в удобную папку (например, на Рабочий стол).
3. Убедиться, что в папке находятся файлы:
    * tea.py
    * gui.py

6. Запуск программы
1. Открыть терминал.
2. Перейти в папку проекта (пример):
cd Desktop/tea-encryption-project-lu
3. Запустить программу командой:
python3 gui.py
После выполнения команды откроется графическое окно программы.

7. Как работать с программой
7.1 Ключ шифрования
* Поле Ключ (hex) уже заполнено автоматически.
* Изменять ключ не обязательно.
* Ключ используется для шифрования и расшифрования данных.

7.2 Шифрование текста
1. В многострочное поле Входной текст ввести текст, который необходимо зашифровать.
2. Нажать кнопку Зашифровать.
3. В нижнем поле появится зашифрованный текст в формате hex.
4. На экране появится сообщение об успешном шифровании.

7.3 Расшифрование текста
1. Вставить зашифрованный текст (hex) в поле Входной текст.
2. Нажать кнопку Расшифровать.
3. В нижнем поле появится исходный текст.
4. На экране появится сообщение об успешном расшифровании.

7.4 Кнопка «Сбросить»
* Кнопка Сбросить очищает все поля ввода и вывода.
* Используется для начала новой операции.

7.5 Завершение работы
* Для выхода из программы достаточно закрыть окно приложения.

8. Возможные ошибки
Неверный формат данных
* При вводе некорректных данных появится сообщение об ошибке.
* Необходимо проверить правильность введённого текста.
Программа не запускается
* Убедиться, что используется команда python3.
* Проверить, что Python установлен корректно.

9. Заключение
Программа tea-encryption-project-lu позволяет выполнять шифрование и расшифрование текстовых данных в удобном графическом интерфейсе без использования командной строки.
