## 1. Цель тестирования

Целью тестирования является проверка корректности работы разработанной программы
шифрования и расшифрования данных на основе алгоритма TEA, а также проверка

Тестирование проводилось в ручном режиме с использованием графического интерфейса

## 2. Среда тестирования

- Операционная система: macOS / Windows
- Язык программирования: Python 3
- Интерфейс: Tkinter (ttk)
- Режим шифрования: ECB
- Алгоритм: TEA (Tiny Encryption Algorithm)

---

## 3. Тестовый план

### Тест 1. Проверка корректности шифрования текста

**Входные данные:**
- Ключ (hex):  
  `00112233445566778899aabbccddeeff`
- Входной текст:  
  `70186736`

**Действия пользователя:**
1. Запустить программу.
2. Ввести ключ шифрования в соответствующее поле.
3. Ввести исходный текст в многострочное поле.
4. Нажать кнопку «Зашифровать».

**Ожидаемый результат:**
- В поле результата отображается зашифрованная строка в шестнадцатеричном виде.
- Появляется диалоговое окно с сообщением об успешном шифровании.

**Фактический результат:**
- Зашифрованный текст успешно отображается.
- Диалоговое окно выводится корректно.

**Результат теста:**  
✅ Тест пройден успешно.

---

### Тест 2. Проверка корректности расшифрования текста

**Входные данные:**
- Ключ (hex):  
  `00112233445566778899aabbccddeeff`
- Зашифрованный текст (hex):  
  `c408dfe5e9ae729083234447e5d31fc1`

**Действия пользователя:**
1. Ввести ключ шифрования.
2. Вставить зашифрованный текст в поле ввода.
3. Нажать кнопку «Расшифровать».

**Ожидаемый результат:**
- В поле результата отображается исходный текст `70186736`.
- Появляется сообщение об успешном расшифровании.

**Фактический результат:**
- Исходный текст восстановлен корректно.
- Сообщение отображается.

**Результат теста:**  
✅ Тест пройден успешно.
