# TEA Encryption Project

Автор: студент Лукьянов Александр Дмитриевич

---

## Описание проекта

Данный проект представляет собой учебную реализацию алгоритма симметричного
блочного шифрования TEA (Tiny Encryption Algorithm).

Алгоритм работает с:
- блоками данных размером 64 бита;
- ключом шифрования длиной 128 бит.

Проект разработан в рамках учебной практики и предназначен
для демонстрации принципов шифрования, дешифрования,
работы с графическим интерфейсом и системой контроля версий Git.

---

## Структура проекта

- tea.py — реализация алгоритма шифрования и расшифрования TEA;
- gui.py — графический интерфейс пользователя (Tkinter);
- instruction.md — подробная инструкция по запуску и использованию;
- tests.md — описание тестирования и отладки;
- deployment.md — рекомендации по внедрению и календарный план;
- README.md — общее описание проекта (данный файл).

---

## Требования для запуска

- Python версии 3.9 или выше
- Операционная система: Windows / macOS / Linux

---

## Запуск программы

1. Откройте терминал.
2. Перейдите в папку проекта.
3. Выполните команду:

   python3 gui.py

После выполнения команды откроется графическое окно программы.

---

## Возможности программы

- Шифрование текстовых данных;
- Расшифрование зашифрованных данных;
- Проверка корректности ключа;
- Обработка ошибок ввода;
- Поддержка копирования и вставки;
- Графический интерфейс пользователя.
